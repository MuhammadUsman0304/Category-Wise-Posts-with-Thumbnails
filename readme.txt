== Description ==

Category Wise Posts with Thumbnails is a Small and fast plugin to display posts category wise in the sidebar, a list of linked titles and thumbnails of  posts category wise

you can add this widget in your any page, also you can customize the number of posts to be displayed in the sidebar, you can change the thumbnail size as well

== Installation ==

1. Upload Category Wise Posts with Thumbnails to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What PHP version are supported / tested? =

Category Wise Posts with Thumbnails works supports starts at PHP 5.3 and is tested on every major release up until PHP 8.1.
